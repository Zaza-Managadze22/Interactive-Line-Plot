# Plot Drawing App

## Description

This app allows users to upload a CSV file and visualize the data as a line plot. Users can control the window of data points displayed and see aggregates for the displayed data.

## Installation

1. Clone the repository.
2. Install dependencies: `npm install`.

## Usage

1. Start the app: `npm run dev`.
2. Upload a CSV file.
3. Adjust the window size and start the animation. For large files and window sizes allow larger interval (T) parameter so the animation can take effect before processing the next window.

## Dependencies

- React
- Chart.js
- PapaParse
